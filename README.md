# mini_ui

Mini UI is a Flutter Design System. A highly customizable flutter ui components.

**Step 1:**
Add `mini_ui : ^1.0.0` to your `pubspec.yaml`

**Step 2:**

```dart
import 'package:mini_ui/mini_ui.dart';
```

**Step 3:**

Boom! Play with Mini UI
